# Text Area

Displays a form textarea or a component that looks like a textarea.

<Canvas of={ComponentStories.Default} />

<Controls />

## Usage

```tsx
import { Textarea } from "mycomponentsui";
```

```tsx
<Textarea placeholder='Type your message here.' className='w-[300px]' />
```

## Examples

### With Label

<Canvas of={ComponentStories.WithLabel} />

```tsx
import { Textarea, Label } from "mycomponentsui";
```

```tsx
<div className='grid w-full gap-1.5'>
  <Label htmlFor='message'>Your message</Label>
  <Textarea placeholder='Type your message here.' id='message' />
</div>
```

### With Text

<Canvas of={ComponentStories.WithText} />

```tsx
import { Textarea, Label } from "mycomponentsui";
```

```tsx
<div className='grid w-full gap-1.5'>
  <Label htmlFor='message-2'>Your Message</Label>

  <p className='text-sm text-muted-foreground'>
```
