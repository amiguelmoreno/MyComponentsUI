# Toggle

A two-state button that can be either on or off.

<Canvas of={ComponentStories.Default} />

<Controls />

## Usage

```tsx
import { Toggle } from "mycomponentsui";
import { Bold } from "lucide-react";
```

```tsx
<Toggle {...args}>
  <Bold className='h-4 w-4' />
</Toggle>
```
